# App de Livros Favoritos - Ionic + Vue.js 3

**Nome completo do aluno:** <Maria Luiza de Freitas>

## Descrição do projeto

Aplicativo mobile desenvolvido para o projeto avaliativo da disciplina.  
O app simula um catálogo de livros favoritos, com telas de login, cadastro, recuperação de senha, listagem de livros e detalhes.  
Os dados são estáticos (mockados) – não há banco de dados nem API.

## Telas desenvolvidas

1. **Login** – campos e-mail/senha, botão Entrar, links para Cadastro e Recuperar senha.
2. **Cadastro** – nome, e-mail, senha, confirmar senha, botão Cadastrar.
3. **Recuperação de senha** – campo e-mail, botão Recuperar Senha.
4. **Home** – lista de livros com título e autor. Ao clicar, navega para a tela de detalhes.
5. **Detalhes do livro** – exibe título, autor e descrição
